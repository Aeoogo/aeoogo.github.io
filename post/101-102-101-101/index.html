查看cpu信息



输出信息如下：

8 Intel(R) Xeon(R) CPU E5506 @ 2.13GHz

说明是8个逻辑cpu


输出信息如下：

4 physical id : 0
4 physical id : 1

说明实际是2个cpu


输出信息如下：

32

说明当前cpu运行在32位下


输出信息如下：

8
